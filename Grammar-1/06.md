# 06.简单句的核心变化 谓语时态【真题演练】
```
The use of this little-known practice `has accelerated` in recent years...
(2019, Reading Comprehension, Part A Text 2)
`has accelerated`： 现在完成时
accelerated: 枷锁
practice：练习，惯例，事情的做法
little-known：很少的知道，不为人所知的，不出名的
well-known: 非常好的知道（出名的）
不为人所知的这样一种做法的使用，在最近几年已经枷锁了。
```

```
Readers `are migrating` away from print anyway.
(2016, Reading Comprehension, Part A Text 4)
`are migrating`: 现在进行时
migrate: 移民，移动走了
读者正在远离印刷业
anyway: 不管怎样
```

```
And a significant number of the homeless `have` serious mental disorders. 许多无家可归的人他们都有严重的精神疾病
(2006, Use of English)
现在的一种常态，一般现在时

serious: 严重的
mental disorders：精神上的疾病
disorder: 无序
homeless: 无家的
home: n. 家
less: 标准的形容词的否定后缀
the homeless: 无家的一类人
a number of : 许多
a significant number of：相当多的数量
```

```
My wife and I `lectured` about this looming danger twenty years ago.
(2001, Reading Comprehension, Part A Text 2)
danger: 危险
looming：模模糊糊，朦朦胧胧，影影约约出现的
lectured: v. 发表演讲，做讲座
在20年前，我和我的妻子关于逼近的危险发表过讲座
```

```
A team of researchers...`would submit` the results of their research to a journal.
(2008, Reading Comprehension, Part A Text 2)
`would submit`: 过去将来时（上下文当中一定有一个过去时）
researcher: 调查研究的人
research：研究
journal：期刊杂志
这组研究人员，他们过去那个时候之后将要提交他们的研究结果给期刊
```

```
In recent years, railroads `have been combining` with each other...
`have been combining`:现在完成进行时
最近的几年，铁路公司现在的之前就在合并，而且还在进行当中...
(2003, Reading Comprehension, Part A Text 3)
```
