# 06.简单句的核心变化 谓语时态【真题演练】
```
The use of this little-known practice `has accelerated` in recent years...
(2019, Reading Comprehension, Part A Text 2)
`has accelerated`： 现在完成时
accelerated: 枷锁
practice：练习，惯例，事情的做法
little-known：很少的知道，不为人所知的，不出名的
well-known: 非常好的知道（出名的）
不为人所知的这样一种做法的使用，在最近几年已经枷锁了。
```

```
Readers `are migrating` away from print anyway.
(2016, Reading Comprehension, Part A Text 4)
`are migrating`: 现在进行时
migrate: 移民，移动走了
读者正在远离印刷业
anyway: 不管怎样
```

```
And a significant number of the homeless `have` serious mental disorders. 许多无家可归的人他们都有严重的精神疾病
(2006, Use of English)
serious: 严重的
mental disorders：精神上的疾病
disorder: 无序
homeless: 无家的
home: n. 家
less: 标准的形容词的否定后缀
the homeless: 无家的一类人
a number of : 许多
a significant number of：相当多的数量
```
